
# Product Scraper

This Project consist the API for Scraping Informations.


## Dependencies

Ruby 3.0.0 \
Rails 7.1.3 \
PostgreSQL 16.1
## Installation


Install gem dependencies
```bash
  bundle install
```

Setup DB
```bash
  rails db:create db:migrate
``` 

running developemnt server
```bash
  rails s
```
